# 🟢 01 — Basics

Beginner-friendly Python scripts covering fundamental concepts.

## Scripts

| File | Concepts Covered |
|------|-----------------|
| `basiccode.py` | Even/Odd, Fibonacci, Leap Year, Palindrome, FizzBuzz, Armstrong, Lambda, Map/Filter |
| `advancecode.py` | Frequency map, word reverse, set intersection, Counter, sorting, two-sum |
| `acronyms.py` | String split, indexing, upper() |
| `bmi_calculator.py` | Float input, arithmetic, if-elif-else |
| `fahrenheit_to_celsius.py` | Functions, formula |
| `email_slicer.py` | String slicing, index() |
| `color_text.py` | colorama library, terminal colors |
| `multiple_input.py` | while loop, break |
| `story_generator.py` | random.choice(), f-strings, lists |
| `dice_roller.py` | random.randint(), input validation |
| `password_generator.py` | random.sample(), string module |
| `password_strength.py` | random.sample(), character pools |

## Run

```bash
python basiccode.py
python bmi_calculator.py
# etc.
```

## Dependencies

```bash
pip install colorama
```
