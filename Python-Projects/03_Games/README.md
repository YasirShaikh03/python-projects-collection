# 🎮 03 — Games
Fun and interactive Python games.
## Scripts
| File | Description |
|------|-------------|
| `animal_quiz.py` | Guess the animal — 3 attempts per question, score tracking |
| `rock_paper_scissors.py` | Rock Paper Scissors vs Computer with score display |
| `chess_game.py` | Full chess with UI — legal moves, check detection, pawn promotion |
## Run
```bash
python animal_quiz.py
python rock_paper_scissors.py
python chess_game.py
```
## Dependencies
`animal_quiz.py` and `rock_paper_scissors.py` — pure Python, no extra dependencies needed!

`chess_game.py` — requires Pygame and python-chess:
```bash
pip install pygame python-chess
```
