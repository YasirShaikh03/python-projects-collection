# 🌐 06 — Web Projects

Web-based applications using HTML, CSS, and JavaScript.

## Files

| File | Description |
|------|-------------|
| `ai_study_planner.html` | AI-powered study planner — single-file, no server needed |

## How to Run

Simply open the file in any browser:

```bash
# Double-click the file, or:
start ai_study_planner.html       # Windows
open ai_study_planner.html        # macOS
xdg-open ai_study_planner.html   # Linux
```

## Features — AI Study Planner

- Add subjects and generate a study plan
- Mark tasks as done with progress tracking
- Weekly goal completion percentage
- Responsive layout (works on mobile too)
- Zero dependencies — pure HTML/CSS/JS, no install needed
