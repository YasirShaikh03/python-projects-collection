## 📂 Project Categories

### 🟢 01 — Basics
Simple Python scripts perfect for beginners.

| File | Description |
|------|-------------|
| `basiccode.py` | Even/Odd, Fibonacci, Palindrome, FizzBuzz, Armstrong & more |

## 🚀 How to Run

```bash

git clone https://github.com/YOUR_USERNAME/Python-Projects.git
cd Python-Projects

# Install dependencies
pip install -r requirements.txt

# Run any script
python 01_Basics/bmi_calculator.py
```
