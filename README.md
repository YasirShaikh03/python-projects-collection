# python-projects-collection
🐍 A collection of 25+ Python mini-projects — from beginner basics to GUI apps, games, utilities, AI/ML (YOLOv8, MediaPipe, OpenCV), and web projects. Built to learn and showcase Python skills.

This repository contains a collection of Python projects ranging from beginner to intermediate and AI-based applications. The goal of this project is to demonstrate practical implementation of Python in areas like automation, GUI development, computer vision, and real-world problem solving.
